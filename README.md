# Postilize homepage redesign

A premium redesign concept for [postilize.com](https://www.postilize.com/), built to compete visually with harvey.ai and legora.com.

- **[index.html](index.html)** — the page. Standalone, no build step: embedded CSS, Google Fonts (Source Serif 4 + Hanken Grotesk), the real Postilize wordmark inlined as SVG. All imagery is gray placeholder boxes labeled with the asset that belongs there.
- **[DESIGN.md](DESIGN.md)** — the token system: Legora-register palette (near-white `#FAFAF9`, ink `#0D1016`, emerald `#005032`), type scale, spacing, component rules.
- **[PRODUCT.md](PRODUCT.md)** — brand voice, audience, and copy principles behind the slimmed messaging.
- **[assets-logo.svg](assets-logo.svg)** — original white wordmark pulled from the live site CDN.

## Preview

Open `index.html` directly in a browser, or:

```sh
python3 -m http.server 4173
```

then visit http://localhost:4173.
